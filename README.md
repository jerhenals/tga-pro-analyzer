# TGA-Pro Analyzer

## 🇪🇸 Descripción
TGA-Pro Analyzer es un software desarrollado para el procesamiento, análisis y visualización de datos termoanalíticos provenientes de ensayos experimentales. El sistema transforma datos crudos en información estructurada mediante la aplicación de métodos matemáticos, filtrado de señales, suavizado y generación de representaciones gráficas técnicas, facilitando el análisis científico y la interpretación de resultados.

## 🇺🇸 Description
TGA-Pro Analyzer is a software developed for processing, analysis and visualization of thermogravimetric data obtained from experimental tests. The system transforms raw data into structured information through the application of mathematical methods, signal filtering, smoothing techniques and graphical outputs, facilitating scientific analysis and interpretation.

## Funcionalidades principales | Main Features
- Procesamiento automatizado de datos termoanalíticos  
- Filtrado y depuración de datos experimentales  
- Aplicación de métodos de suavizado (media móvil, Savitzky–Golay)  
- Cálculo de variables analíticas como conversión y DTG  
- Generación de gráficas técnicas y comparativas  
- Exportación de resultados en formatos Excel, CSV y PNG  

## Arquitectura del sistema | System Architecture
El software está estructurado en módulos que ejecutan el procesamiento de datos en etapas secuenciales: lectura y validación del archivo, depuración de datos, procesamiento matemático, cálculo de variables analíticas, generación de resultados gráficos y exportación.

The software is structured in modular stages that execute data processing sequentially: file reading and validation, data filtering, mathematical processing, analytical calculations, graphical output generation and data export.

## Uso del software | Software Usage
El software permite cargar archivos de ensayos termoanalíticos, configurar parámetros de análisis, generar curvas de conversión y DTG, comparar múltiples ensayos y exportar resultados para informes técnicos, artículos científicos y aplicaciones académicas.

The software allows users to load thermogravimetric test files, configure analysis parameters, generate conversion and DTG curves, compare multiple experiments and export results for technical reports, scientific publications and academic use.

## Finalidad | Purpose
El sistema está orientado a facilitar el análisis termoanalítico en entornos académicos y de investigación, reduciendo la intervención manual, mejorando la reproducibilidad y garantizando consistencia en los resultados.

The system is designed to support thermogravimetric analysis in academic and research environments by reducing manual intervention, improving reproducibility and ensuring consistency in results.
